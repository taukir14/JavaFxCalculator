# 🧮 JavaFX Calculator

A simple and interactive calculator application built using **JavaFX**.  
This project performs basic arithmetic operations such as addition, subtraction, multiplication, and division through a clean graphical user interface.

---

## ✨ Features
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🔄 Clear input functionality  
- 📟 Real-time display output  

---

## 🛠️ Technologies Used
- **Java**
- **JavaFX**

---

## 🖥️ UI Components
- `Stage` – Main window  
- `Scene` – Container for UI  
- `GridPane` – Layout structure  
- `Button` – User input  
- `TextField` – Display screen  

---

## ⚙️ How It Works
The calculator uses **event handling** to detect button clicks.  
Each input is processed and displayed instantly, providing a smooth and user-friendly experience.

---

## 🎯 Objective
This lab project helps in understanding:
- GUI development using JavaFX  
- Event-driven programming  
- Basic application logic design
